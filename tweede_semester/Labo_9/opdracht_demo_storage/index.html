Opdracht 2

Blijft de data behouden als je de pagina herlaadt?

Nee, de data gaat verloren.

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

Nee, hierbij gaat het ook verloren.

Gebruiken twee tabbladen die dezelfde pagina tonen, dezelfde data?

Nee, het wordt alleen op het tabblad getoond waarop de knop wordt ingedrukt.

Opdracht 5.2.1

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

Nee, het gaat verloren.

Blijft de data behouden als je de pagina herlaadt?

Ja, de data wordt achterliggend behouden en begint verder te tellen vanaf de eerste keer dat je opnieuw klikt.

Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?

Nee, per tabblad begint het opnieuw.

Denk je dat een andere pagina (andere file of url) aan de data kan komen als je ze in hetzelfde tabblad opent?

Ja, wanneer het in hetzelfde tabblad geopend wordt wel.

Opdracht 5.3.1

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

Ja, bij het opnieuw klikken telt hij verder.

Blijft de data behouden als je de pagina herlaadt?

Ja, hetzelfde antwoord als hierboven.

Blijft de data behouden als je de browser afsluit, opnieuw start en de pagina weer opent?

Ja, hij telt gewoon weer verder.

Blijft de data behouden als je de pagina in een andere browser opent?

Nee, dan begint hij opnieuw.

Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?

Ja, en ze tellen op elkaar verder.

Denk je dat een andere pagina (andere file of url) aan de data kan komen?

Ja, omdat ze dezelfde sleutel gebruiken om aan de data te komen.
